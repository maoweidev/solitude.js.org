fancybox 和 mediumZoom 二选一，一定要关闭 fancybox 才能开启 mediumZoom
::

```yaml [_config.solitude.yml]
# image lightbox
lightbox: true # 是否开启图片灯箱

# fancybox
# https://fancyapps.com/fancybox/
fancybox: true

# mediumZoom
mediumZoom: false
```
